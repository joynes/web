ChatGPT API Key Integrity Policy for InstantAI

Purpose:
To assure users of InstantAI that their ChatGPT API keys are stored with utmost security and confidentiality directly on their devices.

Scope:
This policy applies specifically to the handling of ChatGPT API keys by the InstantAI available on Google Play.

Policy:

1. Local Encryption and Storage:
- InstantAI stores users' ChatGPT API keys exclusively on the user's device.
- All ChatGPT API keys are stored in an encrypted format, employing industry-standard encryption algorithms, to ensure the key's security and confidentiality.

2. No External Sharing:
- InstantAI does not transmit, share, or backup users' ChatGPT API keys outside the user's device for any reason.

3. User Responsibility:
- Users are encouraged to periodically change their API keys and to ensure their devices have appropriate security protections.

4. Transparency and Trust:
- We value our users' trust and are committed to maintaining the privacy and integrity of their ChatGPT API keys at all times.
